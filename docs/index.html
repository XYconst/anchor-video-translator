01 · How it works

Drop a clip, pick markets, walk away.

Step · 01

Upload the source

Drag an MP4 onto the studio. Any aspect ratio, any duration up to ~5 minutes works comfortably.

Step · 02

Pick the markets

32 languages preloaded. EU-7 preset in one click, or hand-pick. Optional script .docx if you've already got translations.

Step · 03

Place the caption

Drag, resize, rotate. The preview is WYSIWYG. Pick font, colour, outline, shadow, glow. The studio shows you the burn-in size live.

Step · 04

Hit Translate

Three jobs run in parallel by default. ElevenLabs clones the speaker. Gemini transcribes and translates. ffmpeg fits audio to cut without dead air.

Step · 05

Download the cuts

Each language drops into a delivery grid with a preview + download link. Final files live under uploads/{job}/{lang}/final.mp4.

Built for ads

Cut-respecting tempo

When the dub is longer, audio speeds up to +20% before the video deforms. When it's shorter, the video clip compresses. Never silence-padded.

02 · What you need before you start

Two keys.
Fifteen minutes.

Key 1 · Required

ElevenLabs

Voice cloning, TTS, vocal isolation.

Paid recommended
  1. 01
    Open elevenlabs.io/sign-up and create an account.
  2. 02
    Subscribe to at least the Creator plan ($22/mo). Free tier doesn't include voice cloning.
  3. 03
    Go to Settings → API Keys, click Create API key, give it any name (e.g. video-translator).
  4. 04
    Copy the value starting with sk_…. Save it somewhere safe.

Budget estimate

One clip × 5 languages eats ~5-15 minutes of dubbing credits and consumes a clone-voice slot per speaker. Creator's 100k characters covers ~10 typical ads.

Key 2 · Required

KIE.ai

Cheap Gemini-3-Flash proxy. Transcription + translation + word timestamps.

Pay-as-you-go
  1. 01
    Open kie.ai and sign up with email or Google.
  2. 02
    Top up at least $5 from the billing page. Gemini-3-Flash calls are sub-cent each; $5 lasts hundreds of clips.
  3. 03
    Open the dashboard, hit API Keys in the sidebar, click Create.
  4. 04
    Copy the resulting key. Save it alongside the ElevenLabs one.

Why KIE and not Google direct?

KIE proxies Gemini at ~50% of Google's list price. No GCP project needed; sign up with an email and you're done.

03 · Get it running locally

Pick a path.
Both lead to the same studio.

Path A: classic command line. Path B: hand it to Claude Code and let the AI do the typing.

System tools

Python 3.12+

macOS: brew install python@3.12

ffmpeg

macOS: brew install ffmpeg

Node 18+

macOS: brew install node

Path A · Manual

Run it from the terminal

~10 min first time

  1. Step 01 · Clone the repo

    git clone https://github.com/XYconst/anchor-video-translator.git && cd anchor-video-translator

  2. Step 02 · Install the backend

    ./setup.sh

    Creates backend/venv, installs Python deps. Takes ~3 min on first run (torch is heavy).

  3. Step 03 · Drop the keys into .env

    cp .env.example .env

    Open .env in any editor and fill in: 

    ELEVENLABS_API_KEY=sk_xxxxxxxxxxxxxxxxx
KIE_API_KEY=xxxxxxxxxxxxxxxxx

  4. Step 04 · Start both servers

    ./start.sh

    Backend boots on :8000, frontend on :3000. Ctrl + C stops both.

  5. Step 05 · Open the studio

    http://localhost:3000 →

Path B · Claude Code (faster)

Let the AI set it up for you

Recommended

Install Claude Code if you don't already have it. Then:

  1. Step 01 · Clone the repo and open Claude

    git clone https://github.com/XYconst/anchor-video-translator.git && cd anchor-video-translator && claude

  2. Step 02 · Paste this into Claude

    Set up the Anchor Video Translator. Read the README to understand the project, then walk me through getting an ElevenLabs API key and a KIE.ai API key. Once I paste them, write them into a .env file at the project root, run ./setup.sh to install the backend, then start both the backend (port 8000) and frontend (port 3000) with ./start.sh. Open http://localhost:3000 in my browser when ready and tell me what to test first.

  3. Step 03 · Follow the prompts

    Claude will ask you to paste keys when it needs them, run the setup script, boot the servers, and open the browser. The whole flow takes about 10 minutes.

04 · When things go sideways

Common issues, fast answers.

setup.sh fails with "python3: command not found"

Python 3.12+ isn't installed. macOS: brew install python@3.12. Ubuntu: sudo apt install python3.12 python3.12-venv. Then rerun ./setup.sh.

"ffmpeg not found" during a translation

Install it system-wide. macOS: brew install ffmpeg. Ubuntu: sudo apt install ffmpeg. Restart the backend afterwards.

Frontend loads but shows "Could not reach the API"

Backend isn't running. In a second terminal: cd backend && ./venv/bin/uvicorn main:app --port 8000. Check the log for missing env vars.

"Unsupported file type: audio/x-wav" during translation

Pull the latest from main, that bug shipped a fix. git pull origin main && ./setup.sh, restart.

"The read operation timed out"

KIE/Gemini was slow this time. The pipeline retries up to 6 times automatically. If it still fails, your KIE balance might be empty — top up and retry.

Captions look one way in the preview, different in the output

Make sure you're on the latest main — preview is WYSIWYG since the May 14 release. git pull origin main, hard refresh the browser.

Container OOMs / your laptop fan screams

Lower concurrent jobs. Edit .env: MAX_CONCURRENT_JOBS=1. Restart the backend. Default is 3 which assumes 16 GB+ RAM.
